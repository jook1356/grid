# 016. 기능 로드맵 및 추천 구현 목록

## 상태
**검토중** (2026-01-25)

## 컨텍스트

CRUD 및 UndoStack 구현 완료 후, 프로젝트 전체를 분석하여 앞으로 구현할 수 있는 기능들을 정리했습니다.

---

## 현재 구현 완료된 기능

| 분류 | 기능 |
|------|------|
| **Core** | GridCore, DataStore, IndexManager, EventEmitter |
| **CRUD** | ChangeTracker, UndoStack, Command 패턴 |
| **처리** | 정렬, 필터, 집계, 피벗 테이블 |
| **렌더링** | 가상 스크롤 (100만 행), DOM 풀링 |
| **상호작용** | 행/셀/범위 선택, 인라인 편집, 컬럼 조작 |
| **고급** | 행 그룹화, 셀 병합, Multi-Row, 행 고정 |

## 추천 구현 순서

### Phase 1: 핵심 UX 완성 (1-2주)

1. **키보드 네비게이션** - 방향키, Tab, Enter
2. **클립보드** - Ctrl+C/V
3. **정렬 UI** - 헤더 클릭 정렬
4. **컨텍스트 메뉴** - 우클릭 메뉴

### Phase 2: Export/Import (1주)

5. **CSV Export**
6. **Excel Export**
7. **파일 Import**

### Phase 3: 에디터 확장 (1-2주)

8. **Dropdown 에디터**
9. **Date Picker**
10. **Autocomplete**

### Phase 4: 필터 UI (1-2주)

11. **필터 팝업 UI**
12. **다중 필터 조합**

### Phase 5: 프레임워크 래퍼 (2주)

13. **React Wrapper**
14. **Vue Wrapper**

## 결론

총 70개의 기능을 12개 카테고리로 분류했습니다.

**즉시 구현 추천**:
- 키보드 네비게이션 (사용성 핵심)
- 클립보드 지원 (Excel 호환성)
- CSV/Excel Export (실무 필수)

**중기 구현 추천**:
- 필터 UI (헤더 팝업)
- 에디터 확장 (Dropdown, DatePicker)
- React/Vue 래퍼

**장기 구현 고려**:
- 트리 그리드
- 서버 사이드 처리
- 플러그인 시스템
